# WhatsApp Chat Analyzer

A powerful data analysis tool built with Streamlit that analyzes exported WhatsApp chats and generates interactive insights.

## Features

- Message statistics
- Monthly timeline
- Daily timeline
- Activity heatmap
- Most active users
- WordCloud generation
- Most common words
- Emoji analysis

## Tech Stack

Python  
Streamlit  
Pandas  
Matplotlib  
Seaborn  
Plotly  

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
